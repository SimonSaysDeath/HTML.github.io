How HTML5 Works

Images:
This is an image:

Code:
If an image is in a subfolder, no dots.
<img src="/Images/Flag.png" alt="Flag" title="Pixelart Canadian Flag, Made by Kieran Dunn." height="100" width="200">
If it is linked from one subfolder to a different one, say, childPages to Images, do the two dots
<img src="../Images/Flag.png" alt="Flag" title="Pixelart Canadian Flag, Made by Kieran Dunn." height="100" width="200">
If the image is in the same folder, do this.
<img src="Flag.png" alt="Flag" title="Pixelart Canadian Flag, Made by Kieran Dunn." height="100" width="200">

This is differently sized.
<img src="../Images/Flag.png" alt="Flag" height="200" width="400">
