Breaks and comments

Breaks and Comments (br,!--)

Breaks are Incredibly important! they allow a line to end.
It's basically the Enter key of html code.
There is a break right here:--->
And here:--->
but not here:---> This is what happens without a break. It uses the same line.
I pressed Enter after the"--->" and the "break." but didn't put a Break.
Breaks look like this: <br>
they don't have a </br> like <p> has </p>

Comments allow you to write text inside the code that aren't shown.

Superscripts and Subscripts

Superscripts and Subscripts (sup,sub)
Scripts like these (<sup>Super</sup> and <sub>Sub<sub>) can make text do things such as this:

WhatTheHeck?

WHATTHEHECK?

WhatTheHeck?

22 = 4
44 = 256
H2O = Water/Dihydrogen Monoxide
H2S = Hydrogen sulfide
Etc.
They look like this: <sup>What</sup><sub>The</sub><sup>Heck?</sup>

<sup><sub>W<sub>H<sub>A<sub>T<sub>T<sub>H<sub>E<sub>H<sub>E<sub>C<sub>K<sub>?</sub></sub></sub></sub></sub></sub></sub></sub></sub></sub></sub></sub></sub>

<sub>What</sub>The<sup>Heck?</sup>

2<sup>2</sup> = 4
4<sup>4</sup> = 256
H<sub>2</sub>O = Water/Hydrogen Oxide
H<sub>2</sub>s = Hydrogen Sulfide
